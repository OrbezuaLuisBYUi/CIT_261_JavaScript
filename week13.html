Linear: Using the same speed to move it.
Ease: Starting slow, faster and to finish is slow.
Ease-in: Starting slow and hold the same speed after that.
Ease-out: Same speed but will finish slowly.
Ease-in-out: Starting and finish slow but the start is a little bit faster.
